installation d'openbsd sans CD

C'est beau l'an 2000. Le cd est appelé à disparaitre, et nos install par ce médium également. Il existe un tas de méthodes, netboot, dd sur une clé usb et tout.
Personnelement, mes CDRW où je faisais amoureusement cdio blank && cdio tao install44.iso viennent de rendre l'âme, et il est donc temps pour moi de m'intéresser à ces 'nouvelles' 'technologies'.

install existante

Sur une install existante, rien de plus simple, on récupère le bsd.rd qui va bien, et on boot dessus (boot bsd.rd au prompt de boot.)
